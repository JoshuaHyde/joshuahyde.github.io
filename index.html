About me

I've spent my career on the operations side of software: standing up Kubernetes clusters from scratch, running them with Argo CD and GitOps on AWS and GCP, and looking after the data stores and dashboards that go with them — MongoDB, Elasticsearch, Grafana, Prometheus.

The same habits carry over to a server of my own, where I host my personal projects: a few automation agents, a game I'm building, and the networking and monitoring that keep it all online. Everything on this page is real and currently running, and I maintain all of it myself.

AI Agent Platform

            
An assistant platform I run on my own VPS. It lives in Discord, keeps
              long-term memory in a local vector index with embeddings computed on the box, speaks through
              a TTS voice, and hands larger jobs off to coding tools when a task calls for it. It has been
              running continuously for months and I use it every day.

Agent Coordination

            
A shared voice and messaging service for the assistants I run. They're
              built on different tools and model providers, but they all communicate through the same
              channels: one place handles voice synthesis, message routing, and presence, instead of a
              separate integration for each.

Workflow Automation

            
Pipelines that pass work between specialized assistants: one plans and
              breaks a task down, others handle the implementation, and results get reviewed before anything
              ships. Scheduled jobs cover the recurring work too, like daily briefings, health checks, and
              content updates.

Mission Control

            
One dashboard for the whole platform: what each agent is doing, token
