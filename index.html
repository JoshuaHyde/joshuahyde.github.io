The operator

Infrastructure is a craft. I've spent my career keeping production systems fast, observable, and boring to operate — Kubernetes clusters I stand up from scratch and run GitOps-style with Argo CD, on AWS and GCP, with data layers in MongoDB and Elasticsearch and dashboards in Grafana wired to Prometheus.

Off the clock, the same discipline runs my own platform: a VPS that hosts AI agents with local vector memory, a procedurally generated game, generative asset pipelines, and the zero-trust network that ties it together. Everything on this page is something I actually run — not a tutorial I followed once.

AI Agent Platform
